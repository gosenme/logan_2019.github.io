go mod

  • go mod init gin-case 初始化

makefile

  • vim makefile

  • 输入内容

gin

  • go get -u github.com/gin-gonic/gin

  • 需要解决问题，拟 挂代理

init

  • export GO111MODULE=on 打开go mod
  • export GOPROXY=https://mirrors.aliyun.com/goproxy/ 设置代理
  • go mod tidy 手动处理依赖关系
  • go mod vendor 生成vendor目录
