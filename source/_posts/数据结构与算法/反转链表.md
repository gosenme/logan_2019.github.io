title: 反转链表

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    """
    None <- 1 <- 2 <- 3 <- 4 <- 5
    b       a
            b    a
    """
    def reverseList(self, head: ListNode) -> ListNode:
        a, b = head, None
        while a:
            b, b.next, a = a, b, a.next
        return b
        a = list()
        reversed(a)
```
