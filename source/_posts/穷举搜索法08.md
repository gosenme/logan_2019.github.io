递归
    
    1. 确定递归子结构
    2. 确定递归退出条件和退出处理
    3. 递归的自我触发机制

    基本原则是能不使用递归就不使用递归，
    毕竟递归方法存在效率和栈的开销问题，
    在搞不清楚问题规模的情况下，
    很容易出现问题。


二叉树的前序遍历
    
    根-->左-->右
    
    
举个简单的例子
```python
def fib(n):
    
    if n <= 1:
        return 1
    else:
        return fib(n-1)+fib(n-2)

```
